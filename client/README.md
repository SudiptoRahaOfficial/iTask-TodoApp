# iTask Todo App

A simple and responsive **Todo App** built with **React.js** and **Tailwind CSS**.

This is my **first project after learning React.js**, created to practice the core concepts of React by building a real-world application. The project focuses on creating a clean user interface and managing application state using modern React techniques.

---

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Responsive user interface
- Built with reusable React components
- Local Storage support

---

## Tech Stack

- **React.js**
- **Tailwind CSS**

---

## What I Learned

While building this project, I practiced:

- React Components
- JSX
- Props
- State Management (`useState`)
- Event Handling
- Conditional Rendering
- Rendering Lists with `map()`
- Component-based Architecture
- Tailwind CSS Utility Classes

The goal of this project is to strengthen my understanding of React fundamentals by building a complete frontend application from scratch. It serves as the first milestone in my journey toward becoming a professional full-stack engineer.
